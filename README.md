# Legacy Path Explorer

AI-powered Wealth Advisor chat interface for Legacy Investing.

## Setup

1. Clone the repository
2. Install dependencies: `npm install` (for Vercel deployment)
3. Copy `.env.example` to `.env` and add your API keys
4. Run locally: `node server.js`

## Environment Variables

- `OPENROUTER_API_KEY` - Your OpenRouter API key for AI chat functionality

## Deployment

This project is configured for deployment on Vercel. The `api/chat.js` file handles serverless API requests.
